ssBrainTrek

3D Nano-Resolution Neuron Cell Model Explorer and Annotation Tool

Find Out More

BrainTrek allows visualization and comparison of different neuronal morphologies.

Designed originally for a large multi-screen array display (CAVE), BrainTrek was created to quickly allow exploration of high-resolution cellular models, and to facilitate understanding and annotation.

However, not everyone owns a CAVE. To increase the accessibility of BrainTrek, a single-screen version (ssBrainTrek) was created. Built in Unity3D, ssBrainTrek is much more versatile -- supporting Windows, MacOS, and Linux, as well as the Oculus Rift.

To date, there are three modules of ssBrainTrek. CellExplorer allows the full volume exploration of our five ages (postnatal day) of neuron models. CellCompare provides a fast and easy environment for human comparison of neuron models. SynapseTagger, the newest module, enables the user to tag and describe the connections between cells. With these tools, neuroscientists can rapidly measure and record useful datasets to be used in basic research.
